# Bad STL Viewer

A really simple STL viewer written in Rust. I wrote this because Clement Hathaway told me it would impress his boss.

![](screenshots/fighter.png)

## Installing

```bash
git clone https://github.com/connorwiniarczyk/bad-stl-viewer.git
cd bad-stl-viewer
cargo install --path .
```

## Running

```bash
cd examples
bad-stl fighter.stl
```
